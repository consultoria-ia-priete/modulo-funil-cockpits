# 🎯 Funil & Cockpits

> Os **funis de vendas** do método — quiz low-ticket, webinar, consultoria e landing page —
> mais os **cockpits** de acompanhamento. Tudo no estilo Alex SSCIA, pronto pra você
> personalizar com a sua marca e subir na sua conta.

Template — gere a sua cópia e instale com o Claude Code, sem programar.

## 🚀 Como começar (3 passos)

1. **`Use this template`** → `Create a new repository`.
2. **Baixe:** `git clone` + `cd`. (Windows? [Tutorial Windows](docs/windows.md).)
3. **Abra o Claude Code e diga `instalar`** (`claude` → **`instalar`**).
   O Claude te ajuda a personalizar (seu pixel, seu domínio, seus produtos), buildar e subir.
   Aula em [`aula/roteiro.md`](aula/roteiro.md).

## ✅ Pré-requisitos

- A **Base** instalada. Idealmente **CRM Funnels** e **Tracking** já conectados (os funis capturam
  lead no seu CRM e disparam Pixel/CAPI) — mas dá pra subir e plugar depois.
- **Node** + conta **Cloudflare** (pro deploy via Pages).

## 🧩 O que vem na caixa

| Funil | Pra quê |
|---|---|
| **Quiz** (low-ticket) | Porta de entrada: captura + qualificação |
| **Webinar** | Inscrição + sala do webinar |
| **Consultoria** | Página da oferta high-ticket |
| **LP** | Landing page de oferta |
| **Cockpits** | Dashboards de acompanhamento do funil (webinar / serviço) |

> Os **escritórios dos agentes** já vêm na Base.

> ⚠️ Os funis vêm com **placeholders** (`SEU-DOMINIO`, `{{META_PIXEL_ID}}`, etc.) no lugar dos
> dados reais. O Claude te guia a preencher os SEUS antes de subir.
